# 💅 Dayıdınç Teoremi Hesaplayıcı

Birinin senden hoşlanıp hoşlanmadığını "bilimsel"(!) yollarla kanıtlayan o meşhur **Dayıdınç Teoremi** artık dijitalde! 

Serra Dayıdınç'ın efsanevi videosundan ilham alınarak geliştirilen bu web uygulaması, crush'ınızın (hoşlandığınız kişinin) hareketlerini matematiksel bir formüle dökerek size acı (veya tatlı) gerçekleri söyler. 

## ✨ Özellikler

- **Modern ve Tatlı Arayüz:** Pastel pembe tonlarında, göz yormayan şık bir tasarım.
- **Akıllı İpuçları:** Değişkenlerin yanındaki `?` ikonlarının üzerine gelerek hangi duruma kaç puan vermeniz gerektiğini öğrenebilirsiniz.
- **Dinamik Sonuçlar:** Çıkan skora göre renk değiştiren ve moral bozan/sevindiren özel sonuç metinleri.
- **Tamamen Client-Side:** Sadece HTML, CSS ve JavaScript kullanılarak yapıldı, anında çalışır.

## 🧮 Teorem Nasıl Çalışır?

Formülümüz şu şekildedir: 
**DT = [(M × A × D) + E] ÷ (G + F - C)**

*Değişkenler (0-10 arası puanlanır):*
* **M (Mimik):** Size karşı takındığı yüz ifadeleri.
* **A (Aktif Tepki):** İletişim kurarkenki aktiflik seviyesi.
* **D (Dikkatli Süzme):** Size bakış atma ve süzme yoğunluğu.
* **E (Enerji):** Sizin için harcadığı enerji miktarı.
* **G (Genel Sosyal):** Herkese karşı genel sosyallik durumu.
* **F (Fırsatçı Olmama):** Sosyal medyayı vb. fırsat bilip bilmeme durumu.
* **C (Çekingenlik):** Genel içe kapanıklık ve adım atamama seviyesi.
